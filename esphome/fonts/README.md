# External Font Dependencies

## materialdesignicons-webfont.ttf

- **Source:** [Templarian/MaterialDesign-Webfont](https://github.com/Templarian/MaterialDesign-Webfont)
- **Download URL:** https://github.com/Templarian/MaterialDesign-Webfont/raw/master/fonts/materialdesignicons-webfont.ttf
- **License:** Apache License 2.0 ([LICENSE](https://github.com/Templarian/MaterialDesign-Webfont/blob/master/LICENSE))
- **Package:** `@mdi/font` (npm: [npmjs.com/package/@mdi/font](https://www.npmjs.com/package/@mdi/font))
- **Icons website:** https://materialdesignicons.com/ (Pictogrammers)

### Updating

To update to a newer version:

```sh
curl -sL "https://github.com/Templarian/MaterialDesign-Webfont/raw/master/fonts/materialdesignicons-webfont.ttf" \
  -o esphome/fonts/materialdesignicons-webfont.ttf
```
